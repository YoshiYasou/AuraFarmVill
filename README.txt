AuraFarmVill

Description:
AuraFarmVill is a web-based system created to help farmers easily monitor and manage their animals’ health. 
It provides a simple way to record animal information, track medical history, and generate health insights.

Features:
Animal Profiles – Save details such as breed, age, and weight.
Health Records – Track illnesses, treatments, and vaccinations.
Simple Interface – Designed to be farmer-friendly and easy to use.


Tech Stack:
Frontend: HTML, CSS, JavaScript (expandable to modern frameworks)
Backend: PHP & MySQL for data handling and storage
